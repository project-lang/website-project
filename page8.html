Create structures and styles for a webpage

Initially my aim is to have a website, possibly with an animated background, but nothing too flashy or garish. Something mellow. No dramatic colours. Nothing with a lot of flashing lights. I will need to have a sidebar, presumably just the one sidebar unless I have a lot of pages to insert. It mustn’t have anything that can disappear e.g. if a web host suddenly went under taking any links or images with it, so everything must link from my folder. I’ll begin with a very basic idea, but then build on that as I start to make it. I would have liked to have a header and possibly a footer as well, both with images.
